Overview

A fully automated web application designed to streamline dormitory and apartment operations. Built to reduce manual administrative tasks, covering everything from tenant onboarding to automated billing and instant notifications.

  • Dashboard & Analytics: Real-time data visualization of room status and financial trends using Chart.js.
  • Tenant & Security Management: Comprehensive tracking of tenant profiles and parking rights. Implemented strict business logic to securely lock security deposits, ensuring they are only processed for return upon official lease termination.
  • Automated PDF Contracts: Instantly generates and stores standardized lease agreements in PDF format (DomPDF).
